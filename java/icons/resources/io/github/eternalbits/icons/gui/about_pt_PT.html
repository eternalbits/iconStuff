Visão geral do programa ▲

"Icons" é basicamente uma versão que permite traduzir do Microsoft Windows ICO para o Apple macOS ICNS e vice-versa.
Atualmente, a aplicação consiste em alternar entre imagens PNG, pois ambas as funções possuem esta ferramenta, o bitmap da Microsoft que está presente no ICO e a compressão da Apple que está presente no ICNS.

Clicando com o botão direito do rato na imagem, será apresentada a seguinte caixa:
Copiar - Significa que a imagem selecionada será copiada
Colar - Significa que a imagem copiada será adicionada
Eliminar - Significa que a imagem selecionada desaparecerá
Atualizar - O ficheiro inteiro será lido novamente
Fechar - O ficheiro inteiro desaparecerá

Caixa de diálogo de configurações ▲

Utiliza um dos dois métodos na caixa de diálogo aberta
O diálogo aberto utilizado tem dois métodos que correspondem ao método open padrão e um método Java. No Windows, o método padrão não pode ser utilizado quando existem ecrãs de resoluções diferentes.
Será apresentado um aviso se tentar utilizar a opção "Guardar"
Notifica-o quando pretende executar a operação Guardar.
Desmarque a opção "Guardar" se estiver a utilizar "Guardar como"
O ficheiro original será lido novamente, como da primeira vez.
Coloque os ícones por ordem decrescente
Utilize o ecrã para organizar os ícones por ordem da maior para a menor resolução.
Avisar se pretende guardar ícones não padrão
O formato ICNS não permite guardar algumas resoluções. Quando não consegue registar uma resolução, ignora-a ou emite um aviso ao utilizador.
Ignorar ícones maiores que 256 pixéis
No formato ICO, recusa-se a guardar ícones maiores que 256 pixels.
Ignorar ícones duplicados
No formato ICO, recusa-se a registar resoluções duplicadas.
