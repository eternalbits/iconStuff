Panoramica del programma ▲

Icons è principalmente una versione che consente di convertire da Microsoft Windows ICO a Apple macOS ICNS e viceversa.
Attualmente, l'applicazione consiste nel passare da immagini PNG a PNG, poiché entrambe le funzioni dispongono di questo strumento: il bitmap di Microsoft presente in ICO e la compressione di Apple presente in ICNS.

Facendo clic con il pulsante destro del mouse sull'immagine verrà visualizzata la seguente casella:
Copia - L'immagine selezionata verrà copiata
Incolla - L'immagine copiata verrà aggiunta
Elimina - L'immagine selezionata scomparirà
Aggiorna - L'intero file verrà riletto
Chiudi - L'intero file scomparirà

Finestra di dialogo Impostazioni ▲

Utilizza uno dei due metodi nella finestra di dialogo Apri
La finestra di dialogo di apertura utilizzata dispone di due metodi che corrispondono al metodo di apertura standard e a un metodo Java. Su Windows, il metodo standard non può essere utilizzato quando sono presenti schermi con risoluzioni diverse.
Apparirà un avviso se si tenta di utilizzare l'opzione "Salva"
Avvisa quando si desidera eseguire l'operazione di salvataggio.
Deselezionare l'opzione "Salva" se si sta utilizzando "Salva con nome"
Il file originale verrà letto di nuovo, proprio come la prima volta.
Ordina le icone in ordine decrescente
Utilizza lo schermo per disporre le icone in ordine dalla risoluzione più alta a quella più bassa.
Avvisa se si desidera salvare icone non standard
Il formato ICNS non consente il salvataggio di alcune risoluzioni. Quando non riesce a registrare una risoluzione, la ignora o visualizza un avviso all'utente.
Ignora le icone più grandi di 256 pixel
Nel formato ICO, rifiuta di salvare icone di dimensioni superiori a 256 pixel.
Ignora le icone duplicate
Nel formato ICO, rifiuta di registrare risoluzioni duplicate.
