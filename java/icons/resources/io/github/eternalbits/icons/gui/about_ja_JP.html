プログラムの概要 ▲

「Icons」は、主に Microsoft Windows ICO から Apple macOS ICNS への変換、およびその逆変換を可能にするバージョンです。
現在、このアプリケーションは PNG 画像間の切り替え機能（このツールは ICO に含まれる Microsoft のビットマップと、ICNS に含まれる Apple の圧縮形式の両方の機能に備わっています）で構成されています。

画像を右クリックすると、次のボックスが表示されます。
コピー - 選択した画像がコピーされます。
貼り付け - コピーされた画像が追加されます。
削除 - 選択した画像が消えます。
更新 - ファイル全体が再読み込みされます。
閉じる - ファイル全体が消えます。

設定ダイアログボックス ▲

開くダイアログボックスで2つの方法のいずれかを使用します
ファイルを開くダイアログには、標準の開く方法とJavaメソッドの2つの方法があります。Windowsでは、解像度の異なる画面がある場合、標準の方法は使用できません。
保存操作を実行する場合に警告を表示します
保存操作を実行する必要があるときに通知します。
アイコンを降順に並べます
画面を使用して、アイコンを最高解像度から最低解像度の順に並べます。
標準以外のアイコンを保存する場合に警告を表示します
ICNS形式では、一部の解像度を保存できません。解像度の登録に失敗した場合は、その解像度を無視するか、ユーザーに警告を表示します。
256ピクセルを超えるアイコンを無視します
ICO形式では、256ピクセルを超えるアイコンは保存されません。
重複するアイコンを無視します
ICO形式では、重複する解像度の登録は拒否されます。
