Обзор программы ▲

Icons — это в первую очередь версия, которая позволяет вам переводить из Microsoft Windows ICO в Apple macOS ICNS и наоборот.
В настоящее время приложение состоит из переключения между изображениями PNG, поскольку обе функции имеют этот инструмент, битовую карту Microsoft, которая присутствует в ICO, и сжатие Apple, которое присутствует в ICNS.

Щелчок правой кнопкой мыши по изображению отобразит следующее поле:
Копировать — означает, что выбранное изображение будет скопировано
Вставить — означает, что скопированное изображение будет добавлено
Удалить — означает, что выбранное изображение исчезнет
Обновить — весь файл будет прочитан снова
Закрыть — весь файл исчезнет

Диалоговое окно настроек ▲

Использует один из двух методов в открытом диалоговом окне
Используемый диалог открытия имеет два метода, которые соответствуют стандартному методу открытия и методу Java. В Windows стандартный метод не может быть использован при наличии экранов с разным разрешением.
При попытке «Сохранить» появится предупреждение
Уведомляет вас, когда вы хотите выполнить операцию сохранения.
Снимите флажок «Сохранить», если вы используете функцию «Сохранить как»
Исходный файл будет прочитан повторно, как и в первый раз.
Расположить значки в порядке убывания
Используйте экран для упорядочивания значков в порядке от самого высокого к самому низкому разрешению.
Предупреждать, если вы хотите сохранить нестандартные значки
Формат ICNS не позволяет сохранять некоторые разрешения. Если ему не удается зарегистрировать разрешение, он либо игнорирует его, либо выдает предупреждение пользователю.
Игнорировать значки размером более 256 пикселей
В формате ICO он отказывается сохранять значки размером более 256 пикселей.
Игнорировать дубликаты значков
В формате ICO он отказывается регистрировать дублирующиеся разрешения.
