Program Overview ▲

"Icons" is primarily a version that allows you to translate from Microsoft Windows ICO to Apple macOS ICNS and vice versa.
Currently, the application consists of switching between PNG images, as both functions have this tool, Microsoft's bitmap that is present in ICO and Apple's compression that is present in ICNS.

Right-clicking on the image will display the following box:
Copy - Means the selected image will be copied
Paste - Means the copied image will be added
Delete - Means the selected image will disappear
Refresh - The entire file will be read again
Close - The entire file will disappear

Settings Dialog Box ▲

Uses one of two methods in the open dialog box
The open dialog used has two methods that correspond to the standard open method and a Java method. On Windows, the standard method cannot be used when there are screens of different resolutions.
Warn if you want to perform the Save operation
Notifies you when you want to perform the Save operation.
Put the icons in descending order
Use the screen to arrange icons in order from highest to lowest resolution.
Warn if you want to save non-standard icons
The ICNS format does not allow saving some resolutions. When it fails to register a resolution, it either ignores it or issues a warning to the user.
Ignore icons larger than 256 pixels
In ICO format, it refuses to save icons larger than 256 pixels.
Ignore duplicate icons
In ICO format, it refuses to register duplicate resolutions.
