程序概述 ▲

“图标”主要允许您将 Microsoft Windows ICO 转换为 Apple macOS ICNS，反之亦然。
目前，该应用程序包含在 PNG 图像之间切换，因为这两个功能都包含此工具，即 ICO 中存在的 Microsoft 位图和 ICNS 中存在的 Apple 压缩。

右键单击图像将显示以下框：
复制 - 表示将复制所选图像
粘贴 - 表示将添加复制的图像
删除 - 表示将删除所选图像
刷新 - 表示将重新读取整个文件
关闭 - 表示将删除整个文件

设置对话框 ▲

在打开对话框中使用两种方法之一
使用的打开对话框包含两种方法，分别对应于标准打开方法和 Java 方法。在 Windows 上，当屏幕分辨率不同时，无法使用标准方法。
如果要执行保存操作，则发出警告
在您想要执行保存操作时通知您。
按降序排列图标
使用屏幕按从最高分辨率到最低分辨率的顺序排列图标。
如果要保存非标准图标，则发出警告
ICNS 格式不允许保存某些分辨率。当它无法注册分辨率时，它会忽略该分辨率或向用户发出警告。
忽略大于 256 像素的图标
在 ICO 格式中，它拒绝保存大于 256 像素的图标。
忽略重复的图标
在 ICO 格式中，它拒绝注册重复的分辨率。
