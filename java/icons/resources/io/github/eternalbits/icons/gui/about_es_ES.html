Descripción general del programa ▲

Iconos es principalmente una versión que permite convertir de Microsoft Windows ICO a Apple macOS ICNS y viceversa.
Actualmente, la aplicación permite cambiar entre imágenes PNG, ya que ambas funciones cuentan con esta herramienta: el mapa de bits de Microsoft presente en ICO y la compresión de Apple presente en ICNS.

Al hacer clic derecho en la imagen, se mostrará el siguiente cuadro:
Copiar: Se copiará la imagen seleccionada.
Pegar: Se añadirá la imagen copiada.
Eliminar: La imagen seleccionada desaparecerá.
Actualizar: Se volverá a leer el archivo completo.
Cerrar: Desaparecerá el archivo completo.

Cuadro de diálogo de configuración ▲

Utiliza uno de los dos métodos en el cuadro de diálogo Abrir
El cuadro de diálogo de apertura tiene dos métodos que corresponden al método estándar y a un método de Java. En Windows, el método estándar no se puede usar cuando hay pantallas con diferentes resoluciones.
Advierte si desea guardar
Le notifica cuando desea guardar.
Ordena los iconos en orden descendente
Use la pantalla para ordenar los iconos de mayor a menor resolución.
Advierte si desea guardar iconos no estándar
El formato ICNS no permite guardar algunas resoluciones. Si no registra una resolución, la ignora o emite una advertencia al usuario.
Ignorar iconos de más de 256 píxeles
En el formato ICO, no se pueden guardar iconos de más de 256 píxeles.
Ignorar iconos duplicados
En el formato ICO, no se pueden registrar resoluciones duplicadas.
