프로그램 개요 ▲

아이콘은 기본적으로 Microsoft Windows ICO에서 Apple macOS ICNS로 또는 그 반대로 변환할 수 있는 버전입니다.
현재 이 애플리케이션은 PNG 이미지 간 전환 기능을 제공하며, 두 기능 모두 이 도구를 사용합니다. ICO에 포함된 Microsoft 비트맵 파일과 ICNS에 포함된 Apple 압축 파일입니다.

이미지를 마우스 오른쪽 버튼으로 클릭하면 다음 상자가 표시됩니다.
복사 - 선택한 이미지가 복사됩니다.
붙여넣기 - 복사한 이미지가 추가됩니다.
삭제 - 선택한 이미지가 사라집니다.
새로 고침 - 전체 파일을 다시 읽습니다.
닫기 - 전체 파일이 사라집니다.

설정 대화 상자 ▲

열린 대화 상자에서 두 가지 방법 중 하나를 사용합니다.
열기 대화상자에는 표준 열기 방법과 Java 방법에 해당하는 두 가지 방법이 있습니다. Windows에서는 해상도가 다른 화면이 있는 경우 표준 방법을 사용할 수 없습니다.
저장 옵션을 사용하려고 하면 경고가 표시됩니다
저장 작업을 수행하려는 경우 알려줍니다.
다른 이름으로 저장을 사용하는 경우 "저장" 옵션의 선택을 해제하세요
처음과 마찬가지로 원본 파일을 다시 읽습니다.
아이콘을 내림차순으로 정렬합니다.
화면을 사용하여 아이콘을 가장 높은 해상도에서 가장 낮은 해상도 순으로 정렬합니다.
비표준 아이콘을 저장하려는 경우 경고합니다.
ICNS 형식은 일부 해상도를 저장할 수 없습니다. 해상도를 등록하지 못하면 해당 해상도를 무시하거나 사용자에게 경고합니다.
256픽셀보다 큰 아이콘 무시
ICO 형식에서는 256픽셀보다 큰 아이콘을 저장할 수 없습니다.
중복 아이콘 무시
ICO 형식에서는 중복 해상도를 등록할 수 없습니다.
