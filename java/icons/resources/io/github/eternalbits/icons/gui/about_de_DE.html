Programmübersicht ▲

„Icons“ ist in erster Linie eine Version, die die Konvertierung von Microsoft Windows ICO in Apple macOS ICNS und umgekehrt ermöglicht.
Derzeit besteht die Anwendung aus dem Wechsel zwischen PNG-Bildern, da beide Funktionen über dieses Tool verfügen: Microsofts Bitmap, das in ICO vorhanden ist, und Apples Komprimierung, die in ICNS vorhanden ist.

Ein Rechtsklick auf das Bild öffnet folgendes Fenster:
Kopieren – Das ausgewählte Bild wird kopiert.
Einfügen – Das kopierte Bild wird hinzugefügt.
Löschen – Das ausgewählte Bild verschwindet.
Aktualisieren – Die gesamte Datei wird erneut gelesen.
Schließen – Die gesamte Datei verschwindet.

Dialogfeld „Einstellungen“ ▲

Verwendet eine von zwei Methoden im Öffnen-Dialogfeld
Der Dialog zum Öffnen verfügt über zwei Methoden: die Standardmethode zum Öffnen und eine Java-Methode. Unter Windows kann die Standardmethode bei Bildschirmen mit unterschiedlichen Auflösungen nicht verwendet werden.
Warnung beim Speichern
Benachrichtigt Sie, wenn Sie speichern möchten.
Symbole absteigend sortieren
Ordnen Sie die Symbole auf dem Bildschirm von der höchsten zur niedrigsten Auflösung an.
Warnung beim Speichern nicht standardmäßiger Symbole
Das ICNS-Format unterstützt das Speichern bestimmter Auflösungen nicht. Wenn eine Auflösung nicht registriert werden kann, wird sie entweder ignoriert oder eine Warnung ausgegeben.
Symbole größer als 256 Pixel ignorieren
Im ICO-Format werden Symbole mit einer Größe von mehr als 256 Pixeln nicht gespeichert.
Doppelte Symbole ignorieren
Im ICO-Format werden doppelte Auflösungen nicht registriert.
