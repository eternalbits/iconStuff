Présentation du programme ▲

« Icônes » est principalement une version permettant de convertir des fichiers Microsoft Windows ICO vers Apple macOS ICNS et inversement.
Actuellement, l'application permet de basculer entre les images PNG, car les deux fonctions disposent de cet outil : le bitmap de Microsoft présent dans ICO et la compression d'Apple présente dans ICNS.

Un clic à droit sur l'image affichera la boîte de dialogue suivante :
Copier : l'image sélectionnée sera copiée
Coller : l'image copiée sera ajoutée
Supprimer : l'image sélectionnée disparaîtra
Actualiser : le fichier entier sera relu
Fermer : le fichier entier disparaîtra

Boîte de dialogue Paramètres ▲

Utilisez l'une des deux méthodes dans la boîte de dialogue d'ouverture
La boîte de dialogue d'ouverture utilisée propose deux méthodes : la méthode standard et une méthode Java. Avec Windows, la méthode standard ne peut pas être utilisée lorsque les écrans ont des résolutions différentes.
Avertir si vous souhaitez enregistrer
Vous êtes averti lorsque vous souhaitez enregistrer.
Classer les icônes par ordre décroissant
Utilisez l'écran pour classer les icônes par ordre décroissant de résolution.
Avertir si vous souhaitez enregistrer des icônes non standard
Le format ICNS ne permet pas d'enregistrer certaines résolutions. Lorsqu'une résolution échoue, il l'ignore ou affiche un avertissement.
Ignorer les icônes de plus de 256 pixels
Au format ICO, il refuse d'enregistrer les icônes de plus de 256 pixels.
Ignorer les icônes en double
Au format ICO, il refuse d'enregistrer les résolutions répétées.
